# small talk

**Talk Less. Connect More.** 話すより、つながる

A landing page for Small Talk, a low-cost prepaid MVNO based in Sydney, Australia — built for Generation Z.

---

## 🎨 Brand Identity

Based on the SmallTalk JP Brand Deck 2026, this landing page follows the official brand guidelines:

### Colors
- **Sumi Ink** `#1C1C1E` - Primary dark
- **Ai Indigo** `#2D3A4F` - Secondary dark
- **Shu Vermillion** `#D94F3B` - Accent red
- **Kinari Cream** `#F5F0E8` - Light backgrounds
- **Suna Sand** `#C4B9A8` - Muted text
- **Matcha** `#7A9A6D` - Accent green

### Typography
- **Georgia** - Headlines & Display text
- **Calibri** - Body & UI copy

### Voice
Calm • Direct • Warm • Considered

## 🚀 Quick Start

1. **Open the landing page**
   ```bash
   open index.html
   ```
   Or use a local server:
   ```bash
   python3 -m http.server 8000
   # Visit http://localhost:8000
   ```

2. **That's it!** The page is ready to view.
